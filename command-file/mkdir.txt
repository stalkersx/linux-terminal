INFO (no-root) :
  |__ command untuk membuat folder baru

COMMAND PENGGUNAAN (file) :
- $ mkdir  nama_folder
  |__ buat 1 folder baru

- $ mkdir /path/path/nama_folder
  |__ buat 1 folder baru didalam folder path

- $ mkdir {nama_folder,nama_folder,nama_folder}
  $ mkdir /path/{nama_folder,nama_folder}
  |__ buat beberapa folder sekaligus

- $ mkdir -m a=rwx nama_folder
  |__ buat folder dengan izin costum (bisa buat folder izin sekaligus)

- $ mkdir -p /nama_folder/nama_folder/nama_folder
  |__ buat folder dalam folder sekaligus
