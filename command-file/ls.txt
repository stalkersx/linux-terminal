INFO (command-file) :
  |__ perintah menampilkan list file dan folder

COMMAND PENGGUNAAN :
- $ ls
  |__ tampilkan list diposisi saat ini

- $ ls /path/path/path
  |__ tampilkan list file/folder tanpa pindah posisi

- $ ls -l /path/path
  |__ tampilkan list file/folder secara detail

- $ ls -a /path/path
  |__ tampilkan list file/folder termasuk yang tersembunyi

- $ ls -la /path/path
  |__ tampikan semua list file/folder tersembunyi dan secara detail

- $ ls -d /path
  |__ tampilkan list hanya folder
  
- $ ls -R /path/path
  |__ tampilkan list folder dalam folder
  
- $ ls -h /path/path
  $ ls -lh /path/path
  |__ tampilkan list file/folder dengan ukuran ke kb,mb,gb
  
- $ ls -s /path/path
  $ ls -sh /path/path
  |__ tampilkan ukuran file/folder
