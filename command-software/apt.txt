  |__ temukan paket dan menampilkan depkrisi

- $ sudo apt show <[package_name]>
  |__ tampilkan rincian paket

- $ sudo apt autoremove
  |__ hapus konfigurasi atau file paket yang tidak dibutuhkan lagi
