INFO (root) :
  |__ kelola wifi via terminal

COMMAND INSTALL :
- $ sudo apt install wireless-tools

COMMAND PENGGUNAAN (network) :
- $ sudo iw list
  |__ tampilkan platforma wifi adapter perangkat

- $ sudo iw dev wlan0 info
  |__ info adafter wifi pada perangkat

- $ sudo iw dev wlan0 set type <[mode]> <[flags]>
  |__ ubah mode(managed,monitor) wifi untuk flags (none,control)

- $ sudo iw dev wlan0 scan
  |__ scan wifi sekitar

- $ sudo iw dev wlan0 link
  |__ check wifi terhubung
  
- $ sudo iw dev wlan1 set channel <[number]>
  |__ setel atau kunci saluran tertentu dalam mode monitor
  
- $ sudo iw dev wlan1 interface add <[newifname]> type managed
  |__ buat interface baru pada perangkat yang sama
  
- $ sudo iw list | grep -A 10 "supported interface modes"
  |__ check wifi mode yang didukung, monitor,manager,master/hotspot

- $ sudo iw list | grep -A 10 "valid interface combinations"
  |__ check wifi bisa buat interface lebih dari 1, #{ managed } <= 1 (ini adalah hasil interface tidak bisa lebih dari 1)
